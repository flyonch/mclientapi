mClientApi тестовый проект
Проект: mClient


Требования
Node.js (версия 18+)
psql latest

Установка и запуск
Склонируйте репозиторий на ваше устройство:

Установите зависимости:
npm i

Заполните файл env необходимыми данными.
Запустите приложение:
npm run dev
